# Diogenes Protocol

Diogenes might say: Trust procedure, not promises.

This repository contains doctrine, templates, tools, and case studies for procedural autonomy, privacy, and institutional accountability.

## Structure

- doctrine/ — core philosophical and operational doctrine
- templates/ — reusable procedural templates (DSAR, escalation, privacy)
- case-studies/ — real-world procedural applications
- tools/ — scripts and utilities

## Purpose

To provide practical methods for forcing alignment between institutional claims and verifiable reality.
