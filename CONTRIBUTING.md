# Contributing

## Scope
This project accepts contributions to:
- doctrine clarity
- templates (jurisdiction-specific variants encouraged)
- tooling (small, composable scripts)
- case-study structure (not personal data)

## Rules
- Do not submit personal data.
- Prefer small, reviewable changes.
- Templates should be factual, jurisdiction-aware, and written for non-experts.
- Include a short rationale in the PR description.

## Template contributions
- Put new templates under templates/<topic>/.
- Use a clear filename and include a one-line purpose at the top.
